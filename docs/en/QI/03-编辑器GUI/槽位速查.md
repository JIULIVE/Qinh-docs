# Editor Slot Quick Reference

> Belongs to: [Editor GUI](概览.md)　·　Related: [Item Editor](物品编辑器.md) · [Action Editor](动作编辑器.md) · [Set Editor](套装编辑器.md)

The **exact slot map** for each editor GUI. All interfaces are 6 rows (54 slots), with MiniMessage titles in Chinese. Slot indices start at 0 (row 1 is 0–8, row 2 is 9–17, and so on).

> This page is for advanced users / troubleshooting who want to know exactly which slot to click. For day-to-day operations, just refer to the diagrams on sub-pages like [Item Editor](物品编辑器.md).

---

## Common Layout Constants

Many list-style interfaces share these slot arrays:

| Name | Slots | Count | Used for |
|---|---|---|---|
| `browserItemSlots` | 10–16, 19–25, 28–34 | 21 | Item / type browser |
| `editionStatSlots` | 19–25, 28–34, 37–43 | 21 | Item editor field rows |
| `handlerSlots` | 10–16, 19–25, 28–34 | 21 | Handler picker |
| `skillSlots` | 10–16, 19–21, 25, 28–34 | 17 | Skill binding (22–24 left empty) |
| `socketSlots` | 10–16, 19–25 | 14 | Gem sockets |
| `refSlots` | 10–16 | 7 | Action handler list |
| `triggerTabSlots` | 0–5 | 6 | Trigger tabs |
