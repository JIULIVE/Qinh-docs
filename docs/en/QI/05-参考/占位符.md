# PlaceholderAPI Placeholders

> Belongs to: [Reference](命令.md)　·　Related: [Gem Sockets](../02-服主指南/宝石孔.md)

Requires **PlaceholderAPI**. Identifier `qinhitems`. The following placeholders are evaluated against the player's **main-hand item** (`GemPlaceholderProvider`).

| Placeholder | Returns |
|---|---|
| `%qinhitems_legendinlay_set%` | Legendinlay Set name |
| `%qinhitems_li_set%` | Same as above (shorthand) |
| `%qinhitems_magicgem_set%` | MagicGem Set name |
| `%qinhitems_mg_set%` | Same as above (shorthand) |
| `%qinhitems_socket_count%` | Number of filled sockets |
| `%qinhitems_sockets%` | Same as above |
| `%qinhitems_socket_total%` | Total number of sockets |
| `%qinhitems_socket_max%` | Same as above |

Evaluation chain: resolve the main-hand item's `legendinlay` / `magicgem` provider → merge socket catalog → count sockets.
