# Glossary

> Belongs to: [Reference](命令.md)　·　Back: [Home](../README.md)

Sorted by pinyin / alphabet — look anything up here whenever you hit an unfamiliar term.

| Term | Meaning | See |
|---|---|---|
| **AP / AttributePlus** | Third-party attribute plugin, QI's numbers backend | [Attributes & Numbers](../02-服主指南/属性与数值.md) |
| **Affix** | Reusable modifier pack (prefix / suffix / Lore) | [Affixes](../02-服主指南/词缀.md) |
| **Assembly** | The process of composing a template + instance + layers into an ItemStack | [Layers & Assembly](../04-开发者/层与装配.md) |
| **base_values** | Core attributes managed by QI, compiled into ap | [Attributes & Numbers](../02-服主指南/属性与数值.md) |
| **Bridge** | Adapter that interprets a Provider payload | [Provider & Bridge](../04-开发者/Provider与桥.md) |
| **Capability** | Functionality supported by a type (SKILL/ATTRIBUTE/GEM_SOCKET…) | [Item Types](../02-服主指南/物品类型.md) |
| **CMD / CustomModelData** | Custom model data | [Resource Pack](../02-服主指南/资源包.md) |
| **Definition (template)** | The item blueprint after a YAML is compiled | [Item Definition](../02-服主指南/物品定义.md) |
| **Fragment** | Reusable item YAML snippet | [Fragments & Templates](../02-服主指南/碎片与模板.md) |
| **Handler** | The effect unit executed after a trigger fires | [Handlers](../02-服主指南/动作系统/处理器.md) |
| **ICVM** | QI's internal standard vocabulary for attributes | [Core Concepts](../01-入门/核心概念.md) |
| **Instance (instance data)** | Mutable state written into a single ItemStack's NBT | [Core Concepts](../01-入门/核心概念.md) |
| **Layer** | A patch appended after creation (socketing / enhancement, etc.) | [Layers & Assembly](../04-开发者/层与装配.md) |
| **LibraryManifest** | Content pack metadata | [Fragments & Templates](../02-服主指南/碎片与模板.md) |
| **Provider** | An external system payload attached to an item | [Provider & Bridge](../04-开发者/Provider与桥.md) |
| **QCL / QinhCoreLib** | Qinhuai's unified low-level library, a hard dependency of QI | [Overview](../01-入门/概览.md) |
| **QS / QinhSkills** | Skill engine, hooked via `qinhskills:cast` | [Integration](../04-开发者/集成.md) |
| **Section** | Reusable Lore block / random pool | [Sections](../02-服主指南/段.md) |
| **Set** | Bonuses activated by the number of pieces worn | [Sets](../02-服主指南/套装.md) |
| **Soulbound** | An item bound to a player and untradeable | [Soulbinding](../02-服主指南/灵魂绑定.md) |
| **Tier (quality)** | Rarity level | [Quality & Display](../02-服主指南/品质与显示.md) |
| **Trigger / Atom** | The player behavior that fires an action | [Triggers](../02-服主指南/动作系统/触发器.md) |
| **Variable** | Item-level dynamic value, rendered via the `{变量}` placeholder | [Variables](../02-服主指南/变量.md) |
| **WriteDomain** | Policy restricting who can change which part of an item | [Layers & Assembly](../04-开发者/层与装配.md) |
| **.qipack** | QI content pack export format | [Import & Export](../04-开发者/导入导出.md) |
| **Pure item-library mode** | Run mode that only crafts items without adding attributes when AP is absent | [Attributes & Numbers](../02-服主指南/属性与数值.md) |
