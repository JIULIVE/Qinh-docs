这是**服主指南（02 区）的首页**。这一区教你把一座遗迹从「磁盘上的一堆文件」一路配成「玩家在世界里撞见、打通、刷层」的完整体验。

## 🗺️ 一座遗迹从无到有

下面这张流程图把整个服主指南的章节串成一条流水线。**从上往下就是你实际配遗迹的顺序**，每一步都给了对应章节链接。

```text
 ① 准备结构本体（方块）
    ├─ 站在世界里框选已有建筑 ──→ /qr pos1 /qr pos2 /qr save
    └─ 把 WorldEdit 的 .schem 导进来 ──→ /qr import
                        │
                        ▼
 ② 写遗迹模板 template.yml（身份证：长啥样 / 用哪个结构 / 在哪生成 / 出什么）
                        │
            ┌───────────┼─────────────┐
            ▼           ▼             ▼
 ③ 配「在哪生成」  ④ 配「玩法层」   ⑤ 配「产出」
   放置档案 Profile   蓝图 Blueprint    战利品 Loot
   generators/*.yml   blueprint.yml     loottables/*.yml
            │           │             │
            └───────────┼─────────────┘
                        ▼
 ⑥ 预览 & 投放：/qr stage 幽灵预览 → confirm；或交给生成总控自然生成
                        │
                        ▼
 ⑦ endgame：玩家通关 → 钥石右键核心 → 注入层数 + 词缀 = 秘境
                        │
                        ▼
 ⑧ 玩家体验：向导罗盘指路、图鉴收集、组队共享进度、bossbar
```

## 🚦 我该从哪开始？

- **手上已经有一座建好的建筑** → 先看 [选区与保存](./selection-save.md) 把它存成模板，再回 [遗迹模板](./ruin-template.md) 完善字段。
- **手上是别人的 `.schem` 原理图** → 看 [结构文件](./structure-files.md) 的 `/qr import` 一节。
- **完全从零、想搞懂全套配法** → 按上面流程图 ①→⑧ 顺序读，从 [遗迹模板](./ruin-template.md) 入手。
- **只想让现成遗迹更肥 / 加 endgame** → 直接跳 [秘境与钥石](./realms-keystones.md) 和 [战利品系统](./loot-tables.md)。

## 🔧 配完别忘了

- 改完任何 YAML（模板 / 蓝图 / 档案 / 战利品表 / 词缀）→ `/qr reload` 生效。
- 想确认遗迹能不能在脚下生成 → `/qr why <模板>` / `/qr gentest`，详见 [诊断与排错](../05-reference/troubleshooting.md)。
- 全部命令、权限、配置项速查 → [命令大全](../05-reference/commands.md) · [config.yml 全配置](../05-reference/config.md)。
