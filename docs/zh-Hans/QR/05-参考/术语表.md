# 术语表

> 所属：[参考手册](./commands.md)　·　相关：[核心概念与架构](../01-getting-started/core-concepts.md) · [FAQ](./faq.md)　·　返回：[首页](../README.md)

按拼音排序，遇到不懂的名词随时查这里。每条一两句释义 + 详解页链接。完整的「配方 → 实例 → endgame」关系图见 [核心概念与架构](../01-getting-started/core-concepts.md)。

| 术语 | 含义 | 详见 |
|---|---|---|
| **锚点（Anchor）** | 一座按模板在世界某坐标**实际生成出来的真实遗迹实例**。记录用哪个模板、坐标、朝向、尺寸、状态（DORMANT/ACTIVE/CLEARED/RECYCLED）。一个模板可生成很多锚点。 | [核心概念](../01-getting-started/core-concepts.md) |
| **词缀（Affix）** | 秘境的 强化修饰，分数量 / 等级 / 环境规则 / 玩家状态 / 战利品五类。受危险预算约束、同组互斥。怪变强=委托 MM 提 level，QR 不算属性。 | [词缀系统](../02-server-guide/affixes.md) |
| **蓝图（Blueprint）** | 叠在结构上的**玩法层**（`blueprint.yml`）：刷怪点 / 击杀目标 / 奖励箱 / 核心 / 机关 / 子结构变体。坐标都相对结构原点。 | [蓝图与目标](../02-server-guide/blueprint-objectives.md) |
| **最终BOSS门（Boss Gate）** | 一种通关方式：累计击杀指定怪物到阈值（`boss-gate.required-kills`）→ 解锁并生成最终BOSS，击败BOSS = 通关。与 `objectives` 阶段制二选一。 | [蓝图与目标](../02-server-guide/blueprint-objectives.md#六bboss-gate最终boss门) |
| **图鉴（Codex）** | 玩家发现遗迹后记录的「攻略卡」收集册（含发现 / 刷怪 / 战利品信息）。可入库跨服共享。 | [向导与图鉴](../02-server-guide/guide-codex.md) |
| **危险预算（Danger Budget）** | 一座秘境能叠加的词缀「危险值」总额（`realm.tiers.danger-budget`）。每个词缀有 `danger`，预算耗尽就不再叠，防地狱叠加。 | [词缀系统](../02-server-guide/affixes.md) |
| **生成总控** | 把配方变成锚点的**调度大脑**：权重竞争抽签、全局密度约束、活世界定时刷新、打通后消退再生。关掉则回退「每模板各自按概率 roll」的旧行为。 | [自然生成与预加载](../02-server-guide/generation-preloading.md) |
| **地基（Foundation）** | 放置结构时在其底部填充的支撑层 + 边缘地形渐变，避免遗迹悬空或与地面有断层。 | [结构文件](../02-server-guide/structure-files.md) |
| **成长度（Growth）** | 玩家 / 队伍的强度刻度，用于战利品缩放与 `min-growth` 门槛。委托 QinhClass / MMOCore，缺失时内置降级。 | [战利品系统](../02-server-guide/loot-tables.md) |
| **指引物 / 向导（Guide Item）** | 右键后罗盘指向**最近同类遗迹**的物品（`qinhruins:guide_<模板id>`，由物品源给予，QR 不内置 give）。配 HUD / 粒子 / 标题。 | [向导与图鉴](../02-server-guide/guide-codex.md) |
| **钥石（Keystone）** | 激活秘境的钥匙。手持右键遗迹核心 → 该遗迹升为对应层数的秘境。可通关掉落（带天梯）或重铸。`/qr keystone give`。 | [秘境与钥石](../02-server-guide/realms-keystones.md) |
| **掩码（Mask）** | 放置时的方块过滤规则。`target-mask`=只在哪些世界方块处粘贴（融进地形不推土）；`source-mask`/`source-skip`=过滤结构里的源方块。 | [结构文件](../02-server-guide/structure-files.md) |
| **标记块（Marker Block）** | 结构里用作放置指令的特殊方块：屏障 BARRIER→放置时清成空气（镂空 / 门洞）；基岩 BEDROCK→保留原地形不覆盖（半埋遗迹）。受 `config.generation.marker-blocks` 开关控制。 | [结构文件](../02-server-guide/structure-files.md) |
| **机关（Mechanism）** | 蓝图里的「触发器 → 动作」，纯世界交互（开门 / 刷伏兵 / 传送 / 给物），**非属性**。6 类触发器 × 多种动作。 | [机关系统](../02-server-guide/mechanisms.md) |
| **放置档案（Profile）** | 「在哪生成」的可复用规则包（`generators/*.yml`）：维度 / 世界 / 群系过滤、生成层、平整度、间距、概率。模板 `generation.profile: <名>` 引用，可局部覆盖。内置 17 个。 | [放置档案](../02-server-guide/placement-profiles.md) |
| **秘境（Realm）** | 把一座**通关遗迹**注入「层数 + 词缀」后的强化态。怪重刷、规则改写、产出升级。**不是新世界**，仍是原地那座遗迹。 | [秘境与钥石](../02-server-guide/realms-keystones.md) |
| **再生 / 消退（Regen）** | 打通遗迹超过 `regen-hours`、附近无人时，快照还原地形、回收锚点、腾出密度名额长新遗迹。需配合 `cleanup.snapshot-restore`。 | [自然生成与预加载](../02-server-guide/generation-preloading.md) |
| **会话（Session）** | 玩家在某座遗迹内的运行态：进 / 出提示、bossbar、组队共享进度。`/qr info` 可看会话占用。 | [组队与会话](../02-server-guide/party-sessions.md) |
| **快照（Snapshot）** | 生成结构**前**对原地形的存档，`/qr remove` 或消退时据此**还原方块**。受 `cleanup.snapshot-restore` 与 `max-snapshot-volume` 控制。 | [自然生成与预加载](../02-server-guide/generation-preloading.md) |
| **结构（Structure）** | 遗迹的**方块本体**，QR 用 Bukkit 原生 `.nbt`（`.schem` 用 `/qr import` 转）。放置时支持掩码 / 地基 / 标记块 / 分帧。 | [结构文件](../02-server-guide/structure-files.md) |
| **模板（Template）** | 一座遗迹的**身份证**（`templates/<id>/`）：用哪个结构、长啥样、在哪生成、出什么战利品、进入门槛。一个模板可生成多个锚点。`_` 开头文件夹被忽略。 | [遗迹模板](../02-server-guide/ruin-template.md) |
| **层数（Tier）** | 秘境的强化等级。层数越高，怪越强、词缀越多、战利品越肥。钥石带层数（1~`realm.tiers.max`）。 | [秘境与钥石](../02-server-guide/realms-keystones.md) |
| **瓦片（Tile）** | 程序化拼接里的小结构单元。多个瓦片按调色板规则拼成大遗迹。 | [程序化生成](../02-server-guide/procedural-generation.md) |
| **调色板（Tile Palette）** | 程序化拼接的规则集，把多个瓦片（小结构）按权重 / 邻接规则拼成大遗迹。`/qr genstruct <palette>`。 | [程序化生成](../02-server-guide/procedural-generation.md) |

## 相关名词速记

- **生成层（layer）**：surface 地表 / underground 地下 / sky 天空 / ocean-surface 海面 / seabed 海底（放置档案字段）。
- **锚点 4 态**：`DORMANT`（休眠）→ `ACTIVE`（激活）→ `CLEARED`（通关）→ `RECYCLED`（回收）。
- **QinhCoreLib（QCL）**：秦淮统一底层库，QR 的**硬依赖**。
- **MythicMobs（MM）**：可选软依赖，怪物来源 + level 缩放（QR 委托它「让怪变强」）。
