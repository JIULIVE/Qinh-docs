## [1.1.0] - 2026-06-28

> 兼容范围扩展到 **Minecraft 1.21.11 ~ 26.2.x**（api-version 仍为 1.21，向上兼容，无需改配置）。

### 新增 — 内容包系统（一键下载 / 安装）

- **`/qr catalog`**：列出内容包目录（`config.yml` → `content.catalog-url` 指向的 YAML，默认指向官方 catalog）。
- **`/qr download <包id>`**：一键下载并安装内容包（.zip）。解压后顶层目录按白名单对应插件数据目录：`templates/ generators/ palettes/ loottables/ blueprints/ sections/ lang/ affixes.yml`，zip 根可放 `pack.yml` 标注 id/name/version。
- **`/qr packs`**：查看已安装的内容包。
- 服主从此不必手动拖文件——遗迹模板 / 生成档案 / 调色板 / 战利品 / 蓝图 / 词缀都能打包分发与一键装。

### 新增 — 生成性能硬化（对标 BetterStructures 后的实测改进）

- **时间预算分帧放置**：新增 `paste-millis-per-tick`（默认 8ms）——大结构按**墙钟时间切片**后台铺设，超过每 tick 预算即让出，自适应负载、不冻服（每 tick 至少保底铺 1024 块）；`spread-place-threshold`（默认 30000 方块）控制何时启用分帧。小结构仍原生单发。
- **邻接重复惩罚**：房间放置档案支持 `repetitionPenalty`（同类瓦片相邻时降低其权重）与 `noRepeat`（同类瓦片禁止相邻），避免「一长串相同房间」的呆板布局。
- **TilePlanner 回溯**：规划房间连接时引入步数预算 + 回溯（`backtrackOnce`），走入死路时退一步换方案，连通性更好、更少断头路。

### 兼容

- 标注兼容 **Minecraft 1.21.11 ~ 26.2.x**。

- 程序化 Roguelike 秘境引擎：遗迹模板 / 瓦片规划 / 锚点放置 / 多种粘贴引擎（原生 / FAWE 桥）。
- 生成层：地表 / 地下 / 天空 / 海底 / 下界感知，放置档案、维度过滤、`/qr scatter` 预生成。
- 玩法层：图鉴 `/qr codex`（发现收集 + 刷怪战利品攻略卡）、词缀系统（命令 / JS / 广播）、秘境与钥石、机关系统、组队与会话。
- 活世界：live-tick、regen 回收、崩服快照体积保护。
- 导入：BetterStructures 景观离线转换。
