# 📚 参考手册

参考手册是 QCL 的速查区：当你不需要从头读教程，只想确认一个术语、一个错误码、或某次更新改了什么时，就来这里。它收录了贯穿整个秦淮 RPG 生态的术语表、按主题归类的高频问答、各子系统的诊断码与错误码总表，以及遵循语义化版本的更新日志。

用法很直接：遇到陌生缩写（QI、QS、QF 之类）先查术语表；安装、物品源、经济、脚本等使用中的疑惑去翻 FAQ；当 `/qcl status` 报红或 API 返回非 OK 的码时，按码号查诊断表对照含义与处理建议；想知道某个引用格式或行为是哪个版本引入的，则看更新日志。

## 本节内容

- 📖 [术语表](./glossary) — QCL 与秦淮生态常用术语的一句话解释，并指向详细页面。
- ❓ [FAQ](./faq) — 按安装、物品源、经济、GUI 脚本、运维、开发者分组的高频问答。
- 🚨 [诊断码与错误码](./diagnostic-codes) — 各子系统状态码与错误码总表，含触发场景与处理建议。
- 📅 [更新日志](./changelog) — 遵循语义化版本（SemVer）的版本变更记录。
