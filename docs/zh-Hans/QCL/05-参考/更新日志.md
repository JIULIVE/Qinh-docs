# 更新日志（Changelog）

> 导航：[文档首页](../README.md) · [目录](../README.md) · [FAQ](./faq.md) · [物品源引用](../02-server-guide/item-source-references.md) · [术语表](./glossary.md)

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)（SemVer）：版本号格式为 `主版本.次版本.修订号`。新增向后兼容的功能时递增**次版本号**，向后兼容的修复时递增**修订号**。

**当前版本：1.1.0**，约 110 个 Kotlin 文件。

---

## [1.1.0] - 2026-06-14

### 新增

- **ItemsAdder 物品源**：引用格式 `ia-命名空间_id` 或 `itemsadder:命名空间:id`。
- **Nexo 物品源**：引用格式 `nexo-id` 或 `nexo:id`。
- 以上两源均为**反射桥**，对应插件未安装时自动跳过（降级，不报错）。
- `ItemSourceType` 枚举新增 `ITEMSADDER` / `NEXO` 两类。
- 所有走 `ItemSourceManager.parseItemReference` 的调用方**自动支持**这两个新源，子插件无需改动代码。
- `plugin.yml` 的 `softdepend` 增加 `ItemsAdder` / `Nexo`。

---

## [1.0.8] - 首个公开版本

### 新增

- **统一物品源**：原版、CraftEngine、MMOItems、NeigeItems、QinhItems、MythicMobs、CustomFishing、MagicGem。
- **统一属性管道**：跨源属性的统一处理。
- **自定义方块桥**（CraftEngine）。
- **ModelEngine 模型桥**。
- **脚本引擎**（GraalVM）。
- **经济桥 / 占位符桥**等外部对接能力。
