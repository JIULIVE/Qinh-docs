# 📚 参考手册

这是你在配置 QinhItems 时反复回来翻查的「速查区」。前面几章带你从安装一路走到进阶玩法，而本节不讲流程、只供检索：命令怎么打、权限怎么发、`config.yml` 每个字段是什么意思、占位符叫什么名字、报错原文对应哪种修法，以及常见疑问与名词解释。

本节按「问题类型」分页：想敲命令查 [命令大全](./commands) 和 [权限节点](./permissions)；改配置查 [config.yml 全配置](./config)（按主题归类）或 [config.yml 原文批注](./annotated-config)（逐行对照默认文件）；不确定某个 YAML 文件管什么，看 [资源文件总览](./resource-files)。遇到红字报错先翻 [校验报错速查](./validation-errors)，定位不到再用 [诊断与排错](./diagnostics) 里的命令深挖。

这些页面**按需查阅、不必通读**。建议先把本页加入书签，遇到具体问题时再点进对应子页定位答案。

## 本节内容

- ⌨️ [命令大全](./commands) — `/qi` 完整命令树，每条命令的用法、参数与所需权限。
- 🔑 [权限节点](./permissions) — 全部权限节点、默认值与父子关系，含灵魂绑定豁免节点。
- ⚙️ [config.yml 全配置](./config) — 主配置每个字段的含义，按主题分段说明。
- 📄 [config.yml 原文批注](./annotated-config) — 默认配置完整原文，逐行对照批注。
- 🗃️ [资源文件总览](./resource-files) — 插件目录下各 YAML / 资源文件的「地图」：谁管什么、去哪看详解。
- 💬 [消息文案](./messages) — 可被服主自定义的提示文案与显示格式，方便汉化改语气。
- 🔣 [占位符](./placeholders) — PlaceholderAPI 占位符清单，标识符 `qinhitems` 及其返回值。
- 📋 [内置内容清单](./bundled-content) — QI 自带的品质、段 / 词缀池、套装完整清单，可直接引用照改。
- 🚨 [校验报错速查](./validation-errors) — 每条校验报错原文、触发原因、是错误还是警告、以及修法。
- 🩺 [诊断与排错](./diagnostics) — 诊断命令输出怎么读，各项指标的含义。
- 📈 [性能与运维](./performance-ops) — 大量物品 / 高人数下的加载、刷新、缓存调优与运维建议。
- ❓ [FAQ](./faq) — 散落各页的易错点汇成问答，按主题分组速查。
- 📖 [术语表](./glossary) — QI 专有名词（定义 / 层 / 桥 / 能力等）一句话解释加跳转。
