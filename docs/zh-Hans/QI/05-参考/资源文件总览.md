# 资源文件总览

> 所属：[参考](./commands.md)　·　相关：[配置文件](./config.md) · [内置内容清单](./bundled-content.md)

`plugins/QinhItems/` 下所有配置 / 资源文件的**地图**：每个文件管什么、结构要点、去哪章看详解。

> 这些 YAML 文件**顶部都自带中文注释**，本页不重复抄原文，只给「哪个文件管什么 + 关键结构 + 跳转」。需要逐行批注的只有 [config.yml 原文页](./annotated-config.md)。

---

## 一、根目录文件

| 文件 | 管什么 | 详解 |
|---|---|---|
| `config.yml` | 主配置（目录 / 写域 / 动作路由 / 宝石桥 / 战斗 / 绑定 / 属性映射） | [配置文件](./config.md) · [原文](./annotated-config.md) |
| `item_types.yml` | 物品类型与能力（见下方完整表） | [物品类型](../02-server-guide/item-types.md) |
| `item_tiers.yml` | 品质 / Tier（权重 / 容量 / 发光） | [品质与显示](../02-server-guide/quality-display.md) · [内置清单](./bundled-content.md#一内置品质item_tiersyml) |
| `enchant_limits.yml` | 附魔总等级上限（按类型 / 品质） | [附魔上限](../02-server-guide/enchant-caps.md) |
| `plugin.yml` | 插件元数据 / 权限 / 依赖 | [权限](./permissions.md) · [配置文件 §plugin.yml](./config.md#pluginyml-元数据) |

## 二、内容目录

|---|---|---|

| `fragments/*.yml` | 碎片（可复用片段） | [碎片与模板](../02-server-guide/fragments-templates.md) |
| `sections/*.yml` | 段 / 词缀池 / 品质池 + affixes 段 | [段](../02-server-guide/sections.md) · [词缀](../02-server-guide/affixes.md) |
| `sets/*.yml` | 套装定义 | [套装](../02-server-guide/sets.md) · [内置清单](./bundled-content.md#四内置套装setssetsyml) |

### 类型字段说明

| 字段 | 含义 |
|---|---|
| `display_name` | 显示名（编辑器 + 结构化 Lore 的 type 行） |
| `icon` | 编辑器图标材质 |
| `display` | 是否在类型浏览器显示 |
| `default_material` | 新建该类型物品的默认材质 |
| `capabilities` | 能力标记，见 [物品类型](../02-server-guide/item-types.md#1-能力typecapability是什么) |
| `aliases` | 别名（类型查找用） |
| `parent` | 父类型（继承 capabilities） |
| `library_category` | 库面板分类 |
