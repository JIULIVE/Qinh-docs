# 权限节点

> 所属：[参考](./commands.md)　·　相关：[命令](./commands.md)

来自 `plugin.yml` 与命令代码。

| 权限 | 默认 | 子权限 | 说明 |
|---|---|---|---|
| `qinhitems.use` | OP | — | 基础命令（list 等） |
| `qinhitems.give` | OP | — | 给物品 |
| `qinhitems.reload` | OP | — | 重载配置 |
| `qinhitems.inspect` | OP | — | 查看信息 / 诊断 |
| `qinhitems.editor` | OP | — | 打开编辑器 GUI |
| `qinhitems.admin` | OP | use / give / reload / inspect / editor | 全管理权限（父节点） |
| `qinhitems.bypass.soulbound` | — | — | 绕过所有灵魂绑定限制（丢弃 / 拾取 / 破坏容器） |

> `qinhitems.admin` 是父权限，授予它即包含 use / give / reload / inspect / editor 全部子权限。
>
> `qinhitems.bypass.soulbound` 的具体节点可在 `config.yml` 的 `soulbound.bypass-permission` 改，见 [灵魂绑定](../02-server-guide/soulbinding.md#4-配置项)。
