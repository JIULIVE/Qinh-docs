# 术语表

> 所属：[参考](./commands.md)　·　返回：[首页](../README.md)

按拼音 / 字母排序，遇到不懂的名词随时查这里。

| 术语 | 含义 | 详见 |
|---|---|---|
| **AP / AttributePlus** | 第三方属性插件，QI 的数值后端 | [属性与数值](../02-server-guide/attributes-numbers.md) |
| **Affix（词缀）** | 可复用的修饰包（前缀 / 后缀 / Lore） | [词缀](../02-server-guide/affixes.md) |
| **Assembly（装配）** | 把模板 + 实例 + 层合成 ItemStack 的过程 | [层与装配](../04-developer/layers-assembly.md) |
| **base_values（基础值）** | QI 管理的核心属性，编译进 ap | [属性与数值](../02-server-guide/attributes-numbers.md) |
| **Bridge（桥）** | 解读 Provider 载荷的适配器 | [Provider 与桥](../04-developer/providers-bridges.md) |
| **Capability（能力）** | 类型支持的功能（SKILL/ATTRIBUTE/GEM_SOCKET…） | [物品类型](../02-server-guide/item-types.md) |
| **CMD / CustomModelData** | 自定义模型数据 | [资源包](../02-server-guide/resource-pack.md) |
| **Definition（定义 / 模板）** | YAML 编译后的物品设计图 | [物品定义](../02-server-guide/item-definition.md) |
| **Fragment（碎片）** | 可复用的物品 YAML 片段 | [碎片与模板](../02-server-guide/fragments-templates.md) |
| **Handler（处理器）** | 触发后执行的效果单元 | [处理器](../02-server-guide/action-system/handlers.md) |
| **ICVM** | QI 内部属性标准词汇表 | [核心概念](../01-getting-started/core-concepts.md) |
| **Instance（实例数据）** | 写在单个 ItemStack NBT 的可变状态 | [核心概念](../01-getting-started/core-concepts.md) |
| **Layer（层）** | 创建后追加的补丁（打孔 / 强化等） | [层与装配](../04-developer/layers-assembly.md) |
| **LibraryManifest（库清单）** | 内容包元数据 | [碎片与模板](../02-server-guide/fragments-templates.md) |
| **Provider（提供者）** | 挂在物品上的外部系统载荷 | [Provider 与桥](../04-developer/providers-bridges.md) |
| **QCL / QinhCoreLib** | 秦淮统一底层库，QI 硬依赖 | [概览](../01-getting-started/overview.md) |
| **QS / QinhSkills** | 技能引擎，接 `qinhskills:cast` | [集成](../04-developer/integration.md) |
| **Section（段）** | 可复用 Lore 块 / 随机池 | [段](../02-server-guide/sections.md) |
| **Set（套装）** | 按穿戴件数激活的加成 | [套装](../02-server-guide/sets.md) |
| **Soulbound（灵魂绑定）** | 绑定到玩家、不可交易的物品 | [灵魂绑定](../02-server-guide/soulbinding.md) |
| **Tier（品质）** | 稀有度等级 | [品质与显示](../02-server-guide/quality-display.md) |
| **Trigger / Atom（触发器 / 原子）** | 触发动作的玩家行为 | [触发器](../02-server-guide/action-system/triggers.md) |
| **Variable（变量）** | 物品级动态值，`{变量}` 占位渲染 | [变量](../02-server-guide/variables.md) |
| **WriteDomain（写域）** | 限制谁能改物品哪部分的策略 | [层与装配](../04-developer/layers-assembly.md) |
| **.qipack** | QI 内容包导出格式 | [导入导出](../04-developer/import-export.md) |
| **纯物品库模式** | 没 AP 时只造物品不加属性的运行模式 | [属性与数值](../02-server-guide/attributes-numbers.md) |
