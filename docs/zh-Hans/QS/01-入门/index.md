# 📖 入门：从零认识并跑通 QinhSkills

本节是 QinhSkills（QS）的起点。如果你是第一次接触 QS，这里会用最短的路径带你搞清三件事：**QS 到底是什么、为什么要在物品和 MythicMobs 之间插这一层、以及怎么亲手造出第一个能放出来的技能。**

这一节面向所有刚上手的读者——无论你是想快速试跑的服主，还是想先吃透架构再动手的开发者。建议先读 [概览](./overview) 建立心智模型，再按 [安装](./installation) → [快速上手](./quick-start) 的顺序跑通整条链路；如果你习惯先懂原理再配置，可以直接跳到 [核心概念](./core-concepts)。

读完本节，你会知道"按下一个键到放出一团火"中间经过了哪些环节，也能独立排查"技能为什么放不出来"这类最常见的问题。

## 本节内容

- 🧭 [概览](./overview) — QS 是什么、四方分工、与 MythicMobs 的关系、三条设计哲学
- 📥 [安装与环境](./installation) — 硬/软依赖、安装步骤、各依赖缺了会怎样、验证与排错
- 🚀 [5 分钟快速上手](./quick-start) — 把自带的 fire_wave 从占位消息变成真火焰，并绑到物品右键放出来
- 🧩 [核心概念与运行管线](./core-concepts) — 7 阶段运行管线、6 状态状态机、两套字段、门控与桥
