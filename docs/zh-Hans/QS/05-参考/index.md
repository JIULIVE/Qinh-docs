# 📚 参考：速查与排错

这一章是 QS 的**速查区**——不教你怎么搭技能，而是回答你上手后会反复翻的那些问题：命令怎么打、权限怎么配、玩家看到的某条提示是哪条逻辑、`/qs cast` 打回来的结果码是什么意思、自带示例技能在哪、性能怎么调、出问题往哪查。所有内容都以 QS **1.0.22** 为准，命令树、结果码、被动种类与源码严格对齐。

平时用得最多的两张表是 [命令与权限](./commands-permissions)（11 个子命令 + 6 个权限节点）和 [结果码 CastResult](./castresult-codes)（14 个结果码）——遇到技能放不出来，先从这两页和 [诊断与排错](./troubleshooting) 的决策树入手；想吃透名词就翻 [术语表](./glossary)；上线前过一遍 [性能与被动限流](./performance-throttling) 的检查清单。
