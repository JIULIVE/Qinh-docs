# PlaceholderAPI 占位符

> 所属：[参考](命令.md)　·　相关：[宝石孔](../02-服主指南/宝石孔.md)

需安装 **PlaceholderAPI**。标识符 `qinhitems`。下列占位符对玩家**主手物品**求值（`GemPlaceholderProvider`）。

| 占位符 | 返回 |
|---|---|
| `%qinhitems_legendinlay_set%` | Legendinlay 套装名 |
| `%qinhitems_li_set%` | 同上（简写） |
| `%qinhitems_magicgem_set%` | MagicGem 套装名 |
| `%qinhitems_mg_set%` | 同上（简写） |
| `%qinhitems_socket_count%` | 已镶嵌孔数 |
| `%qinhitems_sockets%` | 同上 |
| `%qinhitems_socket_total%` | 总孔数 |
| `%qinhitems_socket_max%` | 同上 |

求值链：解析主手物品的 `legendinlay` / `magicgem` provider → 合并孔目录 → 统计孔数。

> 🖼️ **[图片占位]** 计分板 / Tab 使用这些占位符的展示　·　建议 `assets/placeholders.png`
