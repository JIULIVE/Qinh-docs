# 命令大全

> 所属：[参考](命令.md)　·　相关：[权限](权限.md)

根命令：`/qinhitems`，别名 `/qi`、`/qitems`。下表为完整命令树。

---

## 基础

| 命令 | 权限 | 说明 |
|---|---|---|
| `/qi` / `/qi help` | — | 帮助菜单 |
| `/qi list [类型]` | `qinhitems.use` | 列出所有物品，可按类型过滤 |
| `/qi give <itemId> [玩家] [数量]` | `qinhitems.give` | 给物品（支持拾取绑定）。itemId 补全自注册表 |
| `/qi create <itemId> [类型] [材质]` | `qinhitems.admin` | 创建物品定义 |
| `/qi drop <itemId> [数量]` | `qinhitems.admin` | 在发送者位置掉落（仅玩家） |

## 检查 / 诊断

| 命令 | 权限 | 说明 |
|---|---|---|
| `/qi inspect` | `qinhitems.inspect` | 详细状态检查 |
| `/qi status` | `qinhitems.inspect` | 简要加载状态（物品 / 问题 / 编译失败） |
| `/qi diagnose` | `qinhitems.inspect` | 完整诊断报告 |
| `/qi problems` | `qinhitems.inspect` | 列出已知问题 |
| `/qi problems summary` | `qinhitems.inspect` | 问题摘要 |

## 编辑器

| 命令 | 权限 | 说明 |
|---|---|---|
| `/qi editor [itemId]` | `qinhitems.editor` | 打开编辑器（无 ID 则类型浏览器） |
| `/qi editor browse` | `qinhitems.editor` | 类型浏览器 |

详见 [编辑器 GUI](../03-编辑器GUI/概览.md)。

## 重载 / 刷新

| 命令 | 权限 | 说明 |
|---|---|---|
| `/qi reload` | `qinhitems.reload` | 重载配置、物品、动作并刷新所有玩家 |
| `/qi refresh` | `qinhitems.admin` | 玩家：刷新自己；控制台：刷新所有在线 |
| `/qi refresh all` | `qinhitems.admin` | 热刷新所有在线玩家物品 |
| `/qi refresh inventory` | `qinhitems.admin` | 仅刷新背包（玩家） |
| `/qi refresh armor` | `qinhitems.admin` | 刷新盔甲 + 副手 |
| `/qi refresh equipment` | `qinhitems.admin` | 刷新所有装备槽 |
| `/qi refresh hand` | `qinhitems.admin` | 仅主手 |

刷新机制见 [随机生成 → 刷新服务](../02-服主指南/随机生成.md#7-物品刷新服务)。

## 灵魂绑定

| 命令 | 权限 | 说明 |
|---|---|---|
| `/qi soulbind [等级]` | `qinhitems.admin` | 绑定主手物品给自己（可指定等级） |
| `/qi unsoulbind` | `qinhitems.admin` | 解绑主手物品（仅玩家） |

见 [灵魂绑定](../02-服主指南/灵魂绑定.md)。

## 随机生成

| 命令 | 权限 | 说明 |
|---|---|---|
| `/qi generate [tier] [类型] [数量]` | `qinhitems.admin` | 随机生成物品进背包 |
| `/qi generate preview [tier] [类型]` | `qinhitems.admin` | 预览不生成 |
| `/qi generate pools` | `qinhitems.inspect` | 列出词缀池 |

见 [随机生成](../02-服主指南/随机生成.md)。

## 导入 / 迁移 / 内容包

| 命令 | 权限 | 说明 |
|---|---|---|
| `/qi imports` | `qinhitems.inspect` | 导入文件数 |
| `/qi imports list` | `qinhitems.inspect` | 列出导入文件 |
| `/qi imports run` | `qinhitems.admin` | 执行导入 |
| `/qi imports dry-run` | `qinhitems.admin` | 预览导入 |
| `/qi migrate` | `qinhitems.admin` | 迁移旧物品源 |
| `/qi migrate dry-run` | `qinhitems.admin` | 预览迁移 |
| `/qi packs` | `qinhitems.inspect` | 包文件数与目录 |
| `/qi packs list` | `qinhitems.inspect` | 列出包文件 |
| `/qi packs export` | `qinhitems.admin` | 导出内容包 |
| `/qi packs import` | `qinhitems.admin` | 导入（= apply/sync） |
| `/qi packs sync` | `qinhitems.admin` | 同步（刷新 + 导出） |
| `/qi packs apply` | `qinhitems.admin` | 应用（= sync） |
| `/qi packs gui` | `qinhitems.editor` | 内容包编辑 GUI（仅玩家） |

见 [导入导出](../04-开发者/导入导出.md)。
