### LEGENDARY / EPIC 完整定义示例

```yaml
LEGENDARY:
  name: '<gold><bold>传说</bold></gold>'
  item-glow:
    hint: false
    color: GOLD
  generation:
    chance: 0.08
    capacity:
      base: 10
      scale: 1.2
      spread: 0.12
      max-spread: 0.35

EPIC:
  name: '<light_purple><bold>史诗</bold></light_purple>'
  unidentification:
    name: '史诗'
    prefix: '<light_purple>'
  generation:
    chance: 0.06
    capacity:
      base: 12
      scale: 1.5
      spread: 0.15
      max-spread: 0.4
```

Tier 字段全表见 [品质与显示](../02-服主指南/品质与显示.md#3-tier-字段表)。

## 二、内置段（sections/example_sections.yml）

### 词缀池（weight_join）

| 段 ID | 类型 | 内容 |
|---|---|---|
| `affix_pool` | weight_join | 锋利(5) / 迅捷(3) / 试炼(2) |
| `weapon_affix_pool` | weight_join | 前缀"锋利的" 后缀"之力"；精致的(4)/冰霜的(3)/烈焰的(2)/神圣的(1) |
| `prefix_pool` | weight_join | 坚固的(3)/敏捷的(2)/生命的(2)/神圣的(1) |
| `suffix_pool` | weight_join | 守护者(3)/疾风(2)/生命(2)/圣光(1) |

```yaml
affix_pool:
  type: weight_join
  amount: 1
  values:
    - "5::<gray>锋利</gray>"
    - "3::<blue>迅捷</blue>"
    - "2::<gold>试炼</gold>"
```

## 四、内置套装（sets/sets.yml）

| 套装 ID | 显示名 | 成员前缀 | 档位 |
|---|---|---|---|
| `warrior_bloodlust` | `<red>血色狂暴</red>` | warrior_helmet/chestplate/leggings/boots/sword + `warrior` 前缀 | 2/3/4/5 件 |
| `mage_arcane` | `<aqua>奥术之翼</aqua>` | mage_robe/hat/staff/amulet | 2/3/4 件 |
| `healer_divine` | `<green>神圣庇护</green>` | healer_crown/robe/chalice | 2/3 件 |
| `craftsman_master` | `<gold>工匠大师</gold>` | craftsman_hammer/goggles/apron | 2/3 件 |

### 战士套装（含属性 / 效果 / 自定义 Lore 格式）完整示例

```yaml
warrior_bloodlust:
  display_name: "<red>血色狂暴</red>"
  icon: NETHERITE_SWORD
  belonging_pieces:
    - "warrior_helmet"
    - "warrior_chestplate"
    - "warrior_leggings"
    - "warrior_boots"
    - "warrior_sword"
    - "warrior"                          # 前缀匹配：warrior_xxx 都属此套装
  lore:
    - "<gray>传说中的战士套装</gray>"
    - "<gray>集齐套装可激活特殊效果</gray>"
  lore_format:
    header: "<dark_gray><strikethrough>───────</strikethrough> <white>{name} <strikethrough>───────</strikethrough></dark_gray>"
    inactive_line: "◇ <gray>[{pieces}] {name} [还需{need}]</gray>"
    active_line: "◆ <yellow>[{pieces}] {name} [已激活]</yellow>"
    lore_line: "  <dark_gray>{lore}</dark_gray>"
    lore_divider: "<dark_gray>─ 套装描述 ─</dark_gray>"
    desc_line: "  <dark_gray>{desc}</dark_gray>"
  bonuses:
    - pieces: 2
      name: "战士之心"
      lore: ["<blue>+5 物理伤害</blue>"]
      attributes:
        物理伤害: 5                       # 名称 = AttributePlus 属性显示名
    - pieces: 3
      name: "血色狂暴"
      lore: ["<blue>+10 物理伤害</blue>", "<red>生命恢复 -3%</red>"]
    - pieces: 4
      name: "狂暴之力"
      lore: ["<blue>累计 +15 物理伤害</blue>", "<green>+100 生命</green>"]
      attributes:
        物理伤害: 10                      # 累计：4 件时与 2 件相加 = +15
        生命: 100
    - pieces: 5
      name: "<red>血色战魂</red>"
      lore:
        - "<blue>+20% 攻击伤害</blue>"
        - "<red>生命上限 -100</red>"
        - "<gold>暴击率 +15%</gold>"
        - "<aqua>攻击速度 +25%</aqua>"
      effects:
        SPEED: 1                          # 急迫 II
        STRENGTH: 0                       # 力量 I
```

> 配套的套装件物品在 `items/misc.yml`（`warrior_*` / `mage_*`），见 [物品示例库](../02-服主指南/物品示例库.md#五逐类注解--miscyml-综合演示)。套装机制见 [套装](../02-服主指南/套装.md)。
