# QinhSkills（QS）

> 秦淮 RPG 生态的**技能引擎**:技能定义与释放、施法管线。

!!! note "文档编写中"
    QS 的完整文档正在整理。下面是概要,细节后续补全。

## 它负责什么

- **技能定义与释放** —— 配置技能、处理施法流程。
- **施法管线** —— 变量通道、目标、被动、准星、条件、充能、冷却组、吟唱(CHANNEL)等运行时机制。
- **承接物品动作** —— QI 物品 / 套装里的 `qinhskills:cast` 会把技能释放请求透传到这里。

## 它不负责什么

物品本身(在 QI)、属性数值落地(在数值后端)。
